Open →
How these fit together: Phase 1 proved the rendering layer is the easy part; Phase 2 chose the paradigms worth building; Phase 3 built each paradigm on one shared synthetic generator — the same shape as the real BACI-derived HHI panel, so swapping in real data means replacing a single function; Phase 4 shows the storytelling form for the public page. Eleven screeners across the family turn every view into citable output. All pages are single self-contained HTML files with no external dependencies, ready to host on the IPM site as-is.
